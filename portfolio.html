Mis Proyectos

Project 1: Codmentor

Descripción: CodMentor es una plataforma educativa innovadora diseñada para mejorar las habilidades de programación, proporcionando ejercicios dinámicos generados mediante la API de OpenAI. Los usuarios pueden escribir y ejecutar código en tiempo real en un editor interactivo, con soporte para múltiples lenguajes de programación. El proyecto fue desplegado con Gunicorn y Nginx, y cuenta con HTTPS seguro gracias a Let's Encrypt.

Live Application Repositorio GitHub

Project 2: KakeboAPP

Descripción: KakeboAPP es una herramienta integral de gestión financiera personal basada en el método tradicional japonés Kakeibo. Ayuda a los usuarios a rastrear ingresos y gastos, establecer metas financieras y analizar patrones de gasto. Desarrollada con Django, fue desplegada en DigitalOcean usando Nginx y Gunicorn, con HTTPS seguro a través de Let's Encrypt.

Live Application Repositorio GitHub

Project 3: VS Code DJTemplates-Autocomplete

Descripción: DJTemplates-Autocomplete es una extensión de Visual Studio Code que mejora la eficiencia al trabajar con plantillas de Django, proporcionando autocompletado para etiquetas y filtros comunes. Desarrollada en TypeScript, esta extensión está disponible en el Marketplace de VSCode y ha sido probada para garantizar la compatibilidad con las últimas versiones de VSCode.
